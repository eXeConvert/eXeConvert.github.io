La versión para usar desde un terminal o para automatizar conversiones

Esta es la versión de terminal o consola de eXeConvert. Se usa desde una ventana de Terminal, PowerShell o cmd y es útil para convertir muchos archivos o automatizar tareas.

Qué descargar

Ubuntu y otros sistemas basados en Debian

Descarga el paquete .deb desde la última release y ábrelo con el instalador del sistema.

Linux genérico

Usa el archivo .AppImage si no quieres instalar un paquete del sistema.

Windows

Descarga el instalador .exe y sigue el asistente.

macOS

Descarga el instalador .pkg y sigue el proceso habitual de instalación.

Importante: la CLI está pensada para terminal. Si prefieres trabajar con una interfaz gráfica, usa la página principal de eXeConvert.

Primer uso

1. Instala la versión adecuada para tu sistema En la release verás varios archivos. Elige el instalador que corresponda a tu equipo.
2. Abre un terminal En Linux o macOS puedes abrir la app Terminal. En Windows puedes usar PowerShell o cmd.
3. Comprueba que funciona Ejecuta execonvert --help. Si aparece la ayuda, ya está listo.
4. Convierte un archivo Escribe el archivo de entrada y el de salida en el mismo comando.

Comandos más habituales

Los nombres de archivo pueden ser rutas completas o relativas. Algunos ejemplos:

execonvert proyecto.elpx proyecto.pdf
execonvert proyecto.elpx proyecto.docx
execonvert proyecto.elpx resumen.md
execonvert apuntes.md apuntes.elpx
execonvert legacy.elp actualizado.elpx
execonvert inspect proyecto.elpx

Si un nombre de archivo o carpeta tiene espacios, ponlo entre comillas. Ejemplo: execonvert "Mi proyecto.elpx" "Mi proyecto.pdf"

Ejemplos útiles

Exportar solo algunas páginas de un proyecto:

execonvert proyecto.elpx seleccion.pdf --pages 1,2.1

Inspeccionar la estructura interna de páginas:

execonvert inspect proyecto.elpx --json

Importar Markdown y convertirlo en un proyecto eXeLearning:

execonvert apuntes.md apuntes.elpx

Convertir un proyecto antiguo .elp directamente a PDF:

execonvert legacy.elp legacy.pdf

Para automatizar

La CLI sirve muy bien para tareas repetitivas: convertir lotes de archivos, integrarla en scripts, ejecutarla desde otras herramientas o lanzarla desde tareas programadas.

Ejemplo sencillo en Linux o macOS:

for f in *.elpx; do
  execonvert "$f" "${f%.elpx}.pdf"
done

Ejemplo sencillo en PowerShell:

Get-ChildItem *.elpx | ForEach-Object {
  execonvert $_.FullName ($_.BaseName + ".pdf")
}
